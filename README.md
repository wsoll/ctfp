- Category
  - Objects
    - e.g.: `A`, `B`, `C`
  - Morphisms/Arrows
    - e.g.; `A->B`, `B->C`
- Abstraction
  - Composition
      - ∀(`f: A->B` & `g: B->C`) ∃(`g∘f : A->C`)
        - g after f order because: `g(f(x))`
      - Associativity
        - `h ∘ (g ∘ f) = (h ∘ g) ∘ f`
      - <img src="pic/associativity.png" alt="associativity diagram" width="300"/>
  - Identity
    - ∀x ∈ Ob(C), ∃ id<sub>X</sub> : X → X
      - C: category
      - Ob: objects
    - left & right identity
      - id<sub>b</sub> ∘ f = f
      - g ∘ id<sub>a</sub> = g
